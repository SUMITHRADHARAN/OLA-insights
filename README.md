# OLA-insights
This Ola Data Analyst Project uses SQL to process 50k+ ride records and Power BI to visualize KPIs. It analyzes booking success, revenue by payment method, and driver performance to uncover actionable trends that optimize fleet operations and enhance customer satisfaction in 2026.
